# Supermarket Sales - Exploratory Data Analysis (EDA)

## ApexPlanet Data Analytics Internship

### Project Overview
This project performs Exploratory Data Analysis (EDA) on the Supermarket Sales dataset using Python. The objective is to clean the data, analyze trends, visualize important insights, and prepare the data for future analysis.

---

## Dataset

- Supermarket Sales Dataset
- Total Records: 1000
- Total Features: 17

---

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook (VS Code)

## Data Cleaning

- Checked missing values
- Checked duplicate values
- Converted Date column to datetime format
- Renamed column names
- Saved cleaned dataset

---

## Exploratory Data Analysis

The project includes:

- Histogram of Sales
- Box Plot of Sales
- Sales by Product Line
- Sales by Gender
- Sales by City
- Correlation Heatmap

---

## Key Findings

- No missing values found.
- No duplicate records found.
- Sales are concentrated in lower and middle ranges.
- A few high-value transactions appear as outliers.
- Product lines contribute differently to total sales.
- Cities show different sales performance.
