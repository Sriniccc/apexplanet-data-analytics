# 📊 ApexPlanet Data Analytics Internship

This repository contains my work completed as part of the **ApexPlanet Data Analytics Internship**. The projects focus on data analysis, SQL, Python, and database integration using real-world datasets.

## 🚀 Internship Tasks Completed

### ✅ Task 1 – Foundational Setup & Exploratory Data Analysis (EDA)

### Objective
Set up the Python data analytics environment, clean the dataset, perform exploratory data analysis, and generate meaningful visualizations.

### Work Completed
- Environment setup using VS Code & Jupyter Notebook
- Data cleaning and preprocessing
- Missing value analysis
- Duplicate value detection
- Data type conversion
- Exploratory Data Analysis (EDA)
- Statistical summary
- Correlation analysis
- Data visualization using Matplotlib and Seaborn
- Exported cleaned dataset

### Key Insights
- Sales distribution shows a right-skewed pattern with a few high-value transactions.
- Product line performance varies significantly across categories.
- Customer ratings generally remain high across all branches.
- Payment methods are distributed relatively evenly among customers.

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
